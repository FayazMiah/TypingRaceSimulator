- Part 1: A console based simulation of a typing race between multiple typists.
- Part 2: A graphical version using Swing, adding real time rendering, performance tracking, and gameplay modifiers.

To run Part 1:
Use IntelliJ or another IDE
Open the Part1 folder as a project.
The main method is located in TypingRace.java, so this file can be run.

To run Part 2
Use IntelliJ or another IDE
Open the Part2 folder as a project
The main method is stored within RaceGui.java, so this program file can be run.
To terminate the program, the windows exit button can be used to break.
